Fetches live flight data using Amadeus API

Sends instant SMS alerts via Twilio API

Stores and manages data through Sheety API

Fully built using Object-Oriented Programming

Uses environment variables for secure credential management

Includes fail-safe mechanism to prevent redundant API calls & errors

Modular structure with multiple classes for readability and maintainability

Completed in ~12 days as part of a capstone project

Designed as a real-world practical solution for travelers
